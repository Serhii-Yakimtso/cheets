CSS-змінні

CSS-змінні - контейнер для зберігання значень, для повторного використання в CSS-документі. Об'ява змінної виконується у селекторі :root, і починається з "--".

:root {
--main-color: #3f51b5;
--titleColor: #black;
}

Використання змінної виконується використанням функціїї var()

.title {
color: var(--main-color);
}
