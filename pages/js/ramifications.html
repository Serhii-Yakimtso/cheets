Розгалудження

Розгалуження використовуються для виконання різного коду в тілі залежно від виконання умови.
Коли інтерпретатор зустрічає інструкцію, він обчислює вказану умову в круглих дужках (condition), перетворюючи її до логічного типу (boolean). Якщо умова (condition) перетворюється до true, виконується блок коду, написаний у фігурних дужках {} — (statement).

Інструкція if

Дозволяє виконати певний блок коду тільки в тому випадку, якщо задана умова істинна (тобто приймає значення true).
Якщо умова перетворюється до false, код у фігурних дужках не буде виконуватися

let price = 0;
const subscription = "pro";

if (subscription = = = "pro") {
price = 100;
}
console.log(price); // 100

Інструкція if...else

Використовується у випадках, коли потрібно задати альтернативне виконання коду, при неможливості виконання умови.
Якщо умова перетворюється до true, то виконається код блоку if у фігурних дужках.
Якщо умова перетворюється до false, то виконається код блоку else.

const grade = 85;

if (grade >= 70) {
console.log("Satisfactorily");
} else {
console.log("Unsatisfactorily");
}

Блок else...if

Використовується у випадках, коли необхідно задати декілька послідовних умов.
Перевірка зупиняється, щойно програма знаходить першу умову, яка задовільняється. При цьому виконується тільки тіло відповідного блоку if або else if.

const grade = 85;

if (grade >= 90) {
console.log("Perfectly");
} else if (grade >= 80) {
console.log("Good");
} else if (grade >= 70) {
console.log("Satisfactorily");
} else {
console.log("Unsatisfactorily");
}

Тернарний оператор

Тернарний оператор — це коротша синтаксична заміна інструкції if...else. Дозволяє виконувати певну частину коду залежно від умови

  • обчислюється умова (вказана до знаку "?");
  • якщо умова true, обчислюється вираз після "?";
  • якщо умова false, обчислюється вираз після ":";
  • значення обчисленого виразу повертається як результат роботи.

Увага! Тернарний оператор рекомендується використовувати у найпростіших випадках операції присвоєння чи повернення.

let type;
const age = 20;

if (age >= 18) {
type = 'adult';
} else {
type = 'child';
}

або тернаний оператор
type = age >= 18 ? 'adult' : 'child';

Оператор switch

Дозволяє виконувати різні дії залежно від значення виразу. Працює:

  • Вираз (expression) в операторі switch суворо порівнюється (===) з усіма значеннями в блоках case.
  • Значення виразу порівнюється з кожним блоком case зверху вниз.
  • Якщо значення виразу відповідає значенню в блоці case, виконується код цього блоку.
  • Коли виконання коду блоку case завершено, необхідно використовувати оператор break, щоб вийти зі switch. Це запобігає виконанню коду в наступних блоках case.
  • Якщо жодне значення не відповідає виразу, виконається код у блоці default (якщо він є; default завжди має бути останнім в коді; використовується лише 1 default в коді).

Якщо break не вказано, виконання коду продовжиться в наступному case і далі. Така поведінка називається "провалюванням" (fall-through). Якщо потрібно, щоб кілька блоків case виконували той самий код, можна опустити оператор break між ними (ця поведінка відображенна у другому прикладі).

const fruit = 'apple';

switch (fruit) {
case 'apple':
console.log('Apple selected');
break;
case 'banana':
console.log('Banana selected');
break;
case 'orange':
console.log('Orange selected');
break;
default:
console.log('The fruit is unknown');
}

const day = 3;

switch (day) {
case 1:
case 2:
case 3:
case 4:
console.log('This is a working day');
break;
case 5:
case 6:
console.log('It is a day off');
break;
default:
console.log('Invalid');
}

Логічні оператори

Логічні оператори використовуються для перевірки множинних умов, наприклад, в інструкціях if.

Перетворення типів означає приведення значення будь-якого типу даних у логічне (булеве) значення true або false.

"І" && Оператор зліва направо перевіряє почергово обидва операнди на істинність та повертає або значення останнього істинного (тільки правого) операнда, або першого хибного (лівого чи правого), на якому він запнувся "hello" && 5; // 5 true

«АБО» || Як тільки оператор знайшов операнд, який перетворюється на true, він зупиняється та повертає його значення. Якщо до істини було перетворено перший операнд, то другий навіть не буде оцінюватися. Якщо всі операнди перетворюються на false, результатом буде значення крайнього правого операнда. true || false; // true true
false || true; // true
5 || false; // 5
false || 5; // 5
"hello" || 0; // "hello"
0 || "hello"; // "hello"
0 || false; // false false
false || 0; // 0
null || ""; // ""
"" || null; // null
Приклад запису: (a < 10 || a > 30)
«НІ» ! Приводить операнд до логічного значення (true або false) і потім заперечує (інвертує) його, тобто заміняє на протилежне: true —> false, а false —> true. Використовується для перевірки від зворотного. !false; // true true
!0; // !0 -> !false -> true
!""; // !"" -> !false -> true
!null; // !null -> !false -> true
!true; // false false
!3; // !3 -> !true -> false
!"Mango"; // !"Mango" -> !true -> false
Приклад запису: isOnline && !isBlocked;
