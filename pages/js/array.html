Масиви

Загальне

Масив — це впорядкована структура даних, яка використовується для зберігання колекції елементів. Можуть містити елементи різних типів даних, таких як числа, рядки, булеві значення тощо. Для створення використовуються квадратні дужки []. Усередині дужок перераховуються елементи масиву, розділені комами.

const mixed = ['apple', 10, true];

Кожен елемент масиву має свій порядковий номер - індекс. Індексація елементів масиву починається з 0. Щоб отримати доступ до елемента масиву, використовується синтаксис квадратних дужок.

const planets = ['Earth', 'Mars', 'Venus'];
console.log(planets[0]); // 'Earth'

Перевизначення значення елемента

const planets = ['Earth', 'Mars', 'Venus'];
planets[0] = 'Jupiter';
planets[2] = 'Neptune';
console.log(planets); // ['Jupiter', 'Mars', 'Neptune']

Довжина масиву

const planets = ['Earth', 'Mars', 'Venus'];
console.log(planets.length); // 3

Індекс останнього елемента

const planets = ["Earth", "Mars", "Venus"];
const lastIndex = planets.length - 1;
console.log(planets[lastIndex]); // "Venus"

Приведення типів

const array = [1, true, "Poly"];
console.log(String(array));// "1,true,Poly"
console.log(array + "5"); // "1,true,Poly5"

console.log(Number([])); // 0
console.log(Number([1])); // 1
console.log(Number([1, 2, 3])); // NaN

console.log(Boolean([])); // true />
console.log(Boolean([1, 2, 3])); // true
