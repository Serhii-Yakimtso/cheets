Визначення та терміни

HTML (HyperText Markup Language — мова гіпертекстової розмітки)

  • Елемент HTML — це набір із тегів та змісту.

  • Тег — це елемент мови розмітки гіпертексту.

  • Атрибут — це додаткове значення, що налаштовує елементи або регулюють їх поведінку.

  • Ідентификатор — це унікальне ім'я елементу.

  • Батьківський елемент — це елемент, в який вкладений інший тег.

  • Дочірній елемент — це елемент вкладенний у інший тег.

  • Сусіди — це елементи зі спільним батьком.

  • Предок — це елемент, в який вкладенні інші теги, другого і більше рівнів вкладеності.

  • Нащадок — це елемент, який вкладений в інший тег, другого і більше рівнів вкладеності.

CSS (Cascading Style Sheets — каскадні таблиці стилів)

  • CSS — це мова для опису та зміни зовнішнього вигляду елементів — оформлення та позиціонування елементів HTML розмітки.

  • Правило — блок CSS-коду, що вказує стилі елементам HTML-розмітки. Складається з селектора і групи оголошень у фігурних дужках.

  • Клас — глобальний атрибут, що дозволяє CSS та JS обрати та отримати доступ до стилю та поведінки елементів HTML. Ім'я класу:

    • задається тільки англійською мовою;
    • лише маленькі літери та тире;
    • це обов'язково іменник, що описує цей елемент.

  • Селектор — вказує браузеру, до яких елементів розмітки потрібно застосувати стилі з цього правила.

  • Оголошення — вказує браузеру, до яких елементів розмітки потрібно застосувати стилі з цього правила.

  • Властивість — описує, який тип стилю необхідно застосувати до елементу HTML.

  • Значення — вказує, який саме стиль необхідно застосувати до елементу HTML.

  • Псевдоклас — ключове слово, додане до селектора, яке визначає його особливий стан (наприклад, :hover).

  • Псевдоелемент — ключове слово, додане до селектора, яке дозволяє стилізувати певну частину вибраного елемента (наприклад, ::after).

  • at-rule (@-правила) — е вирази CSS, які показують CSS, як поводитися. Вони починаються зі знака «@», за яким слідує ідентифікатор, і включають все до наступної крапки з комою «;» або наступного блоку CSS, залежно від того, що настане раніше.

  • Flexbox — це CSS-модуль, який визначає набір властивостей для розміщення, вирівнювання та розподілу простору між елементами в контейнері.

  • Позиціонований елемент - елемент, значення властивості position якого відрізняється від static

  • Трансформований елемент — це елемент, що має будь-яке значення властивості transform, відмінне від none
