Позиціонування

Позиціонування в CSS дозволяє контролювати розміщення елементів на сторінці, визначаючи їх позицію відносно інших елементів, контейнерів або вікна браузера.

За позиціонування відповідає властивість position.

Позиціонований елемент - елемент, значення властивості position якого відрізняється від static

Опис Властивість Значення Примітка
тип позиціонування елемента position static Статичне позиціонування (за замовчуванням)
relative Відносне позиціонування - елемент залишається у потоці сторінки. Візуально елемент можна змістити щодо його первісного положення. Використовується лише у зв'язці з абсолютним позиціонуванням (елемент не вилучається з потоку сторінки, продовжуючи займати місце)
absolute Абсолютне позиціонування - елемент вилучається з потоку сторінки. Точка відліку - найближчий елемент-предок, у якого властивість position має будь-яке значення крім static, або від <body>
fixed Фіксоване позиціонування - вилучається із потоку сторінки та позиціонується відносно в'юпорту
sticky Липке позиціонування -
  • Поки елемент знаходиться у своєму контейнері, і контейнер не виходить з області видимості при вертикальному прокручуванні — елемент поводиться як позиціонований відносно (relative).
  • Як тільки частина батьківського контейнера виходить із області видимості, елемент фіксується на сторінці до тих пір, поки хоча б якась частина контейнера присутня на екрані.
  • Якщо весь контейнер вийде з області видимості, то знову увімкнеться поведінка як при відносному позиціонуванні — елемент зникне з екрану за контейнером.
У липкого елемента має бути задане положення, наприклад, top: 0. Елемент-контейнер має бути більшим за висотою, ніж липкий елемен
налаштування положення елемента при позиціонуванні top px, % відстань від верхнього краю
left px, % відстань від лівого краю
bottom px, % відстань від нижнього краю
right px, % відстань від правого краю
глибина (положення) елемента на осі Z від очей користувача до екрану z-index число
  • Чим більше значення z-index всередині цього контексту, тим вище розташовується елемент.
  • Мінімальне (навіть негативне) значення z-index говорить про те, що інші елементи поза потоком будуть розташовані вище цього елементу.
  • Якщо у двох елементів однакові значення z-index, вищим буде елемент, оголошений останнім у HTML-коді.

2D-трансформації

2D-трансформації дозволяють змінювати вигляд елементів на сторінці без зміни їх початкового розташування в потоці документа, додаючи динаміки та інтерактивності.

Трансформований елемент — це елемент, що має будь-яке значення властивості transform, відмінне від none

За трансформацію відповідає властивість transform.

transform-origin - властивість, що визначає початок перетворень елемента. в якості значень використовуються top, right, bottom, left та значення у будь-яких одиницях виміру. Якщо використати три значення - зміщення буде відбуватись у трьох координатах (X, Y, Z). За замовчуванням початок перетворень знаходиться в центрі елемента

transform-origin: 2px;
transform-origin: bottom;
transform-origin: right top;
transform-origin: left 2px;
transform-origin: right bottom 2cm;

Переміщають елемент у горизонтальному (вісь X) та/або вертикальному (вісь Y) напрямках щодо початкового положення.

CSS-переходи

CSS-переходи дозволяють додавати анімаційні ефекти до змін CSS-властивостей, таких як колір, розмір, положення або прозорість. Вони створюють плавні переходи між початковим і кінцевим станом властивостей.

За переходи відповідає властивість transition.

Все, що необхідно зробити для налаштування переходу, це:

  • вказати початкові значення властивостей, що анімуються, і значення властивостей переходу;
  • при події (наприклад, :hover), задати нові значення властивостей, до яких необхідно анімувати елемент.

.box {
    transition-property: background-color, transform;
    transition-duration: 500ms;
    transition-timing-function: linear;
    transition-delay: 0;
}

.box:hover {
    background-color: teal;
    transform: rotate(180deg);
}

CSS-анімації

CSS анімації дозволяють створювати плавні переходи та рухи елементів на сторінці без використання JavaScript.

Властивість animation - визначає анімацію для елемента. Складається з кількох параметрів: імені анімації, тривалості, таймінгу, затримки та кількості повторів.

.my-element {
    animation: move 2s ease-in-out infinite;
}
  • move — це ім'я анімації (визначається за допомогою @keyframes);
  • 2s — тривалість анімації (2 секунди);
  • ease-in-out — функція таймінгу (початок та кінець повільний, а середина швидша);
  • infinite — анімація повторюватиметься безкінечно.

@keyframes - визначає зміни властивостей на різних етапах анімації. У ньому описуються ключові моменти (відсотки) від початку до кінця анімації.

Ключові слова: from (0%) - початок анімації, а to (100%) - її кінець.

@keyframes move {
    0% {
        transform: translateX(0);
    }
    50% {
        transform: translateX(100px);
    }
    100% {
        transform: translateX(0);
    }
}
