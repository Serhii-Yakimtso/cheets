Одиниці виміру
Одиниця Назва Опис Приклад
px Pixels Абсолютна одиниця вимірювання, яка використовується для надання фіксованого розміру HTML-елементам. Один піксель – найменша можлива одиниця на екрані, що має окреме значення кольору. Може бути дробовим. 250px
% Percentages Відносна одиниця вимірювання, розміри елементів залежать від властивостей їх батьківського елемента. 50%
em Em

Одиниця вимірювання, яка наслідує своє значення відносно розміру шрифту батьківського елемента.

Розмір елементу = значення (em) * Розмір шрифту батьківського елементу

20em
rem Rem

Одиниця вимірювання, яка наслідує своє значення відносно розміру шрифту кореневого елемента, для веб-сторінки таким елементом є тег html.

Розмір елементу = значення (rem) * Розмір шрифту html елемента

11rem
vw View Width

Одиниця вимірювання, яка наслідує своє значення відносно ширини видимої частини веб-сторінки.

1vw (view width) – 1% ширини вікна

Ширина елементу = значення (vw) * Розмір (ширина) вікна перегляду

50vw
vh View Height

Одиниця вимірювання, яка наслідує своє значення відносно висоти видимої частини веб-сторінки.

1vh (view height) – 1% висоти вікна

Висота елементу = значення (vh) * Розмір (висота) вікна перегляду

50vh
vmin Vmin Одиниця вимірювання, яка визначається як мінімальний розмір екрана між шириною та висотою, і становить 1% від меншого з двох розмірів екрана. 50vmin
vmax Vmax Одиниця вимірювання, яка визначається як максимальний розмір екрана між шириною та висотою, і становить 1% від більшого з двох розмірів екрана. 50vmax

    content: "→";

@-правила
Правило Опис
@import Використовується для імпортування правил стилів з інших таблиць стилів. Правило має бути визначене у верхній частині таблиці стилів, перед будь-якими іншими @-rules (крім @charset та @layer) та оголошеннями стилів, інакше воно буде проігноровано.
@keyframes Керує проміжними кроками в послідовності анімації CSS, визначаючи стилі для ключових кадрів (або маршрутних точок) напротязі послідовності анімації.
@layer Використовується для оголошення каскадного шару, а також дозволяє встановлювати пріоритети при наявності декількох каскадних шарів.
@media

Використовується для застосування чистини стилів при виконанні одного або декількох медіа-запитів.

@media screen and (min-width: 900px) and (max-width: 1599px) {}

all - Підходить для всіх пристроїв.
print - Призначена для посторінкових матеріалів і для документів, що проглядаються на екрані в режимі попереднього перегляду друку.
screen - Призначений в першу чергу для кольорових екранів комп'ютерів.
speech - Призначений для синтезаторів мови.

@charset Визначає кодування символів, що використовується в таблиці стилів.
@page Використовується для зміни деяких властивостей CSS під час друку документа (margin, orphans, widows, і розриви сторінок документів).
@supports Прив'язує набір вкладених правил у CSS-блоці, обмеженому фігурними дужками, до умови, яка є одним або декількома тестовими оголошеннями у вигляді пар property:value, об'єднаних разом операторами and, or not.

CSS-змінні

CSS-змінні - контейнери для зберігання значень, для повторного використання в CSS-документі. Об'ява змінної виконується у селекторі :root, і починається з
"--".

:root {
    --main-color: #3f51b5;
    --titleColor: black;
}

Використання змінної виконується використанням функціїї var()

.title {
    color: var(--main-color);
}
