Таблиці

Теги
Тег Опис Батько
table Основа таблиці
caption Опис (підпис) таблиці table
colgroup Групування одного або декількох стовпців таблиці. Дозволяє задавати спільні стилі або визначати атрибути групі стовпців. Ставиться після елемента <caption> (якщо використовується), але перед елементами <thead>, <tbody>, <tfoot> і <tr>. table
col Використовується всередині елемента <colgroup> для визначення стилів або атрибутів окремих стовпців таблиці. colgroup
thead Заголовок (шапка) таблиці table
tbody Тіло таблиці (може бути декілька у таблиці) table
tfoot Набір рядків, що сумують інформацію стовпців таблиці table
tr Елемент визначає ряд комірок у таблиці thead, tbody, tfoot
th Елемент визначає комірку як заголовок групи комірок таблиці та може використовуватися як дочірній елемент елемента <tr> tr
td Елемент комірки таблиці даних tr
Атрибути
Атрибут Опис Сумісний тег
colspan Вказує скільки (число) стовпців необхідно об'єднати td, th
rowspan Вказує скільки (число) рядків необхідно об'єднати (якщо поставити 0, тоді його дія поширюється до кінця табличної секції - <thead>, <tbody>, <tfoot>) td, th, tr
span Кількість стовпців на які поширюється одна стилізація або атрибут colgroup, col
abbr Скорочення довгих змістів комірок. Якщо заголовок стовпця або рядка є довгим, можна надати скорочену версію, яка буде використовуватися при озвучуванні або відображенні в інтерфейсах, де немає місця для повного контенту/змісту (мобільні екрани).
Покращення доступності. 		td, th
scope Визначення області дії заголовка <th>. Він допомагає вказати, які комірки даних <td> належать до певного заголовка.
Значення:
col — заголовок стосується всього стовпця.
row — заголовок стосується всього рядка.
colgroup — застосовується до заголовків, які стосуються групи стовпців.
rowgroup — застосовується до заголовків, які стосуються групи рядків.
Покращення доступності. 		th
headers Використовується в <td> для зв'язку з відповідними заголовками таблиці <th>.
В <th> вказується id, в <td> у атрибуті headers вказуються значення атрибуту id відповідного елементу, що відповідає заголовку таблиці для конкретної комірки даних.
Покращення доступності. 		td

Властивість Опис Значення Примітка

separate межі комірок залишаються окремими

fixed ширина стовпців визначається лише першим рядком таблиці (заголовком), і не змінюється на основі вмісту
caption-side Визначає розташування опису (підпису) таблиці top зверху
bottom знизу
