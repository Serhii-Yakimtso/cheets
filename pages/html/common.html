@-правила
Правило Опис
@import Використовується для імпортування правил стилів з інших таблиць стилів. Правило має бути визначене у верхній частині таблиці стилів, перед будь-якими іншими @-rules (крім @charset та @layer) та оголошеннями стилів, інакше воно буде проігноровано.
@keyframes Керує проміжними кроками в послідовності анімації CSS, визначаючи стилі для ключових кадрів (або маршрутних точок) напротязі послідовності анімації.
@layer Використовується для оголошення каскадного шару, а також дозволяє встановлювати пріоритети при наявності декількох каскадних шарів.
@media

Використовується для застосування чистини стилів при виконанні одного або декількох медіа-запитів.

@media screen and (min-width: 900px) and (max-width: 1599px) {}

all - Підходить для всіх пристроїв.

screen - Призначений в першу чергу для кольорових екранів комп'ютерів.

CSS-змінні

CSS-змінні - контейнери для зберігання значень, для повторного використання в CSS-документі. Об'ява змінної виконується у селекторі :root, і починається з
"--".

:root {
    --main-color: #3f51b5;
    --titleColor: black;
}

Використання змінної виконується використанням функціїї var()

.title {
    color: var(--main-color);
}

Векторна графіка

SVG (Scalable Vector Graphics) — формат графіки та мова для опису векторних зображень.

Переваги

  • масштабування — на відміну від растрової графіки, SVG не втрачає якості при масштабуванні
  • розмір файла — векторні зображення важать набагато менше растрових, якщо використовуються для іконок або іншої абстрактної графіки. Використання вектора для фотореалістичних зображень навпаки веде до великого розміру файла.
  • динамічність — за допомогою CSS і JavaScript можна змінювати параметри векторного зображення, наприклад, колір або рамки

Застосування

  • тег img - іконки, для яких не потрібно додавати ефект ховеру або фокусу.
  • властивість background-image - абстрактний фон
  • тег svg ("вбудований SVG" (inline svg)) - іконки, для яких потрібно динамічно змінити колір або інші характеристики векторного зображення з CSS
Властивість Опис Значення Примітка
fill колір заливки ключове слово red
RGB rgb(255, 0, 0);
RGBА rgba(255, 0, 0, 0.3)
HEX #ff0000
stroke колір обводки те саме що й для fill
stroke-width шириниа контуру обводки те саме що й для fill

SVG-спрайт - збирання усіх файлів іконок до одного svg файлу. При доєднанні файлу до проекту, необхідно його перейменувати на sprite.svg.

<svg>
    <use href="../../images/sprite.svg#ідентифікатор"> </use>
</svg>

Усі вектору у спрайті записані через тег <symbol> та індентифіковані. Щоб звернутись до певного вектору треба вказати шлях

Стилізація посилань

Псевдоелементи

  • :link - стан за замовчуванням, у якому знаходиться посилання, коли воно не перебуває в жодному іншому стані;
  • :visited - посилання, після відвідування (існує в історії браузера)
  • :hover - посилання при наведенні на нього курсору миші;
  • :focus - посилання, при фокусуванні (за допомогою клавіатури);
  • :active - посилання, при активуванні.

Властивості

  • color — колір тексту;
  • cursor — стилізація курсору миші при наведені;
  • outline — стиль контуру тексту;
  • text-decoration — нижнє підкреслення

Фон елемента

  • Контентні зображення - несуть смислове навантаження, відносяться до вмісту сторінки та допомагають донести до користувача корисну інформацію. Додаються в HTML-розмітці за допомогою тега img (фотографія продукту на картці товару в інтернет-магазині; фотографії членів команди; логотип компанії; рекламні банери; карти з локацією; картинки слайдера).
  • Декоративні зображення - використовуються для додаткового оформлення. Вони не несуть корисної інформації та є елементами дизайну. Додаються через CSS як фон елемента або, у випадку векторної графіки, розмічаються в HTML тегом svg (кастомні маркери списків; допоміжні іконки; плашки; фонові зображення блоків).
