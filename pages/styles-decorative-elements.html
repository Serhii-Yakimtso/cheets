Фон елемента

  • Контентні зображення - несуть смислове навантаження, відносяться до вмісту сторінки та допомагають донести до користувача корисну інформацію. Додаються в HTML-розмітці за допомогою тега img (фотографія продукту на картці товару в інтернет-магазині; фотографії членів команди; логотип компанії; рекламні банери; карти з локацією; картинки слайдера).

  • Декоративні зображення - використовуються для додаткового оформлення. Вони не несуть корисної інформації та є елементами дизайну. Додаються через CSS як фон елемента або, у випадку векторної графіки, розмічаються в HTML тегом svg (кастомні маркери списків; допоміжні іконки; плашки; фонові зображення блоків).

Векторна графіка

SVG (Scalable Vector Graphics) — формат графіки та мова для опису векторних зображень.

Переваги

  • масштабування — на відміну від растрової графіки, SVG не втрачає якості при масштабуванні
  • розмір файла — векторні зображення важать набагато менше растрових, якщо використовуються для іконок або іншої абстрактної графіки. Використання вектора для фотореалістичних зображень навпаки веде до великого розміру файла.
  • динамічність — за допомогою CSS і JavaScript можна змінювати параметри векторного зображення, наприклад, колір або рамки

Застосування

  • тег img - іконки, для яких не потрібно додавати ефект ховеру або фокусу.
  • властивість background-image - абстрактний фон
  • тег svg ("вбудований SVG" (inline svg)) - іконки, для яких потрібно динамічно змінити колір або інші характеристики векторного зображення з CSS

SVG-спрайт - збирання усіх файлів іконок до одного svg файлу. При доєднанні файлу до проекту, необхідно його перейменувати на sprite.svg.

Усі вектору у спрайті записані через тег <symbol> та індентифіковані. Щоб звернутись до певного вектору треба вказати шлях

<svg>
<use href="../images/sprite.svg#ідентифікатор"> </use>
</svg>

Псевдоелементи

Псевдоелементи - використовуються для додавання декоративних ефектів без необхідності створення додаткових порожніх тегів у розмітці. За замовчуванням — це рядкові елементи.

  • ::before - створює псевдоелемент перед всім вмістом елемента (на початку)
  • ::after - створює псевдоелемент після всього вмісту елемента (наприкінці).

Властивість content дозволяє додати текстовий контент всередину псевдоелемента - обов'язкова властивість. Якщо за макетом текстовий контент не потрібний, його значенням необхідно поставити порожній рядок

У селекторі можна використовувати лише один псевдоелемент, який має додаватись після простого селектора (тега, класу, ідентифікатора)

Приклад використання псевдоелементу із псевдокласом

.box:hover::after {
content: '';
}
